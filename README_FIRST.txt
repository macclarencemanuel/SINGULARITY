Project Title:
SINGULARITY

Playable Game Link:
https://macclarencemanuel.github.io/SINGULARITY/

Important:
Please use the playable GitHub Pages link above to run the game.

The full source code is included in the Source_Code folder.

Opening index.html directly may not work on some browsers because the project uses JavaScript modules, JSON data, videos, audio, and browser security may block local file loading through file://.

Recommended Way to Run:
1. Open the GitHub Pages link.
2. Wait for the intro video to finish.
3. Click the title screen.
4. Go to settings and adjust the volume slider to your preference
5. Click Play.
6. Read the How to Play screen.
7. Click Start Game.
8. After finishing the intro, wait for the assets to load. Then enjoy.

Included Files:
- Source code of the entire project
- Integrated instructions in the game
- User manual
- Flowchart guide
- Member list with contribution percentage

Controls:
Move: W A S D
Run: Hold Shift
Aim: Mouse
Shoot: Left Click
Pause: Esc
Open Shop/Terminal: Press T when prompted

Game Objective:
Clear each room by defeating all enemies.
Collect credits and healing items.
Use the terminal shop to upgrade the player.
Defeat the bosses in Room 5 and Room 10.
Escape after Room 10 to complete the game.
